#  GitHub Actions — Reusable Workflows & Job Reporting

This repository contains examples and notes about **advanced GitHub Actions concepts** — focusing on **Reusable Workflows** and **Job Reporting**, which are essential for scalable and maintainable CI/CD pipelines.

---

##  What Are Reusable Workflows?

Reusable workflows allow you to **create one workflow file** that can be **used by multiple repositories or other workflows** — making your CI/CD setup **modular and DRY (Don’t Repeat Yourself)**.
